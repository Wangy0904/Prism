# 📐 PRISM: 基于语义折射的跨界产品创新引擎 🌈



## 🌟 项目亮点 (Key Features)
一个基于 AI 提示词驱动的工业设计发散系统，旨在通过三阶段 AI 管线辅助设计师完成从原始意图到创新方案的跨界迁移。
⚡ Semantic Leap: 突破设计固着，实现跨维度灵感发散
“用算法打破认知边界，从语义深处挖掘最远距离的‘跨界’刺激。”
- 打破设计固着：利用 SBERT 嵌入模型 计算源产品与候选物之间的语义距离，帮助设计师强制逃离“逻辑舒适区”，获取跨度极大的非相关领域刺激物。
- 多维度属性解构：针对选定的刺激物，系统自动从功能、交互、感知、情感四个底层维度进行结构化解构，提取最具迁移价值的设计基因。支持对四个维度的属性进行单点重新生成，精准控制发散的“深度” 。

🧠 Logic Bridge: 场景驱动迁移，构建“意料之外”的设计逻辑
“拒绝生搬硬套，为抽象属性寻找最契合的场景土壤与用户痛点。”
- 场景化能力迁移：系统不再进行简单的形态拼凑，而是通过 AI 深度分析，将远距离属性转化为源产品的“新能力”，并推演出具体的使用场景。
- 隐性痛点洞察：自动识别用户在特定场景下的交互瞬间，精准定义目标人群及其未被察觉的痛点，为跨界属性的植入提供合理的逻辑支点。

🏗️ Structural Synthesis: 从创意火花到结构化设计方案的终极跃迁
“告别模糊概念，一键产出包含结构、功能与感官反馈的完整设计定义。”
- 全方案自动构建：基于前期的灵感与场景设定，系统自动生成包含产品功能、物理结构、材料说明及交互反馈机制在内的标准化设计方案。
- 感官闭环反馈：特别强化了对“人机交互闭环”的设计，定义产品如何通过视觉、听觉、触觉等感官维度给予用户实时的状态反馈。


## 快速开始 (Getting Started)
1. 后端配置 (Backend Setup)
- cd backend
- python -m venv venv
- source venv/bin/activate  # Windows 使用 venv\Scripts\activate
- pip install -r requirements.txt
- 在 .env 中配置您的 LLM_API_KEY
- python app.py
2. 前端配置 (Frontend Setup)
- cd frontend
- npm install
- npm start
🔄系统交互流程 (Workflow)
- 输入与发散：设计师输入产品意图或图片，系统生成跨界联想物。
- 解构与对齐：系统将联想物转化为功能、交互、感知、情感四个维度的属性，并进行语义距离排序。
- 推演与渲染：基于所选属性完成场景推演，并最终生成具体的设计方案与 AI 效果图。
